 Context-Aware Conversational Chatbot

 Overview
This project is a simple context-aware chatbot built using Python for customer support use cases.  
It can understand user queries, maintain conversation context, and provide appropriate responses across multiple steps.



Features
- Context memory (remembers previous user intent)
- Intent recognition using keyword matching
- Multi-turn conversation handling
- Coherent and logical dialogue flow
- Lightweight (no external libraries required)


 Tech Stack
- Python 3
